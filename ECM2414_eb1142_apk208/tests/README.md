# ECM2414_PairCoursework

To run the file Java 17 must be installed
This can be checked with java -version

to run the .jar cards file:
use: java -jar cards.jar

to run the tests Java 17 and Maven 3.9.10 must be installed
use: mvn -version
and to install:
sudo apt update
sudo apt install maven


to run the tests all libraries are included, use the command below
use: cd cardgame/
and then use: mvn test
